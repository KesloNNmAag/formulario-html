Perguntas e respostas do item 2

1-Na tag "input" com atributo type="text", para que serve o atributo "placeholder"?

Uma área com um texto reservado

2-Na tag "input" com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

Para definir altura e largura do elementos input

3-Na tag input para que serve o atributo "value"?

Define valor inicial no campo de entrada

4-Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez??

Seria a type="radio". A compilação rádio é uma maneira para se limitar a opção do usuário.

5-Para que serve a tag "label"? Para que serve o atributo "for" nessa tag?

É usada para definir uma legenda para que o usuário tenha acesso a um formulário.

6-Na tag "option" para que serve o atributo "selected"?

Serve para definir uma pré-escolha pelo usuário

7-Nas tags de formulário para que serve o atributo "required"?

É um atributo em formulário para que o usuário não deixe de preencher algo antes de mandá-lo

8-Na tag "form" para que serve o atributo "action"?

O formulário,quando preenchido, os dados devem ser mandados para um arquivo no servidor,mas nesse caso,é mandado para um arquivo PHP.

9-Na tag "form" para que serve o atributo "method"?

Para onde o formulário deve ser mandado após preenchido.Metedo padrão para enviar é get. method="get"

10-Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag "form"?

Atributo Novalidate

11-Qual valor deve ser informado no atributo "method" na tag "form" em um formulário que será usado para fazer upload de arquivos?

O atributo method tem dois valores, GET e POST,para mandar grande quantidade de dados ultilize o POST que é ainda mais seguro que o GET que envia os dados por URL sendo limitado a quantidade
